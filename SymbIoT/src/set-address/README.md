#set-address

This program sets the address of a node in EEPROM.
Upon starting the sketch, enter a hexadecimal digit from 0 to f.
The node's **octal** adddress will be assigned as follows:

- 0: 00
- 1: 01
- 2: 02
- 3: 03
- 4: 04
- 5: 05
- 6: 011
- 7: 021
- 8: 012
- 9: 022
- a: 013
- b: 023
- c: 014
- d: 024
- e: 015
- f: 025
